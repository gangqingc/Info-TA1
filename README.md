# Info-TA1
Le joueur controle la direction d'avancement du serpent par le clic gauche du souris. 
Le serpent doit slalomer entre les bords de l'écran. 
Pour le jeu il faut que le joueur fasse manger à son serpent plus de pastilles qui s 'estime officiellement sur l'écoulement. 
Et le serpent s'allonge à chaque fois que la taille du pastille est atteinte. 
Alors que le serpent avance inexorablement avec une vitesse propotionnelle avec une taille, 
le joueur ne peut pas être suivi par une direction à suivre, 
soit que la tête du serpent ne touche pas les murs ou son propre corps, qu'il risque de mourir.
